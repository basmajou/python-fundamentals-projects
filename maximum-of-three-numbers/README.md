# Maximum of Three Numbers
Determines the maximum value among three given numbers.

## Key features:
- Uses conditional logic to compare numbers and identify the largest value.
- Handles potential input errors.


## Example Input 
```bash
Enter 1st number: -6
Enter 2nd number: 10
Enter 3rd number: 80
```
## Example Output 

```bash
Max number is: 80
```
